# MARKET PREDICTION MACHINE LEARNING #

Model used: LSTM (Long Short-Term Memory) a variant of Recurrent Neural Network

Learning Rate & Num of Epochs is tuned based on the stock META. Requires re-tuning of LR & Epochs in other stocks

Project uses: UV, JupyterLab, VS code

# To run project on JupyterLab: #

## Run on Terminal ##
uv run JupyterLab
